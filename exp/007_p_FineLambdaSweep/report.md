# Experiment 007: Fine Lambda Sweep

## 实验目的

在有效区间 {0, 3e-5, 1e-4, 3e-4, 1e-3} 细扫 lambda，找 Pareto 最优点（alignment vs reconstruction trade-off）。

## 实验设置

| 参数 | 值 |
|---|---|
| 黑盒模型 | GPT-2 (768-dim) |
| 激活层 | `transformer.h.6`, `transformer.h.11` |
| SAE 架构 | 同 002 |
| Lambda 值 | 0, 3e-5, 1e-4, 3e-4, 1e-3 |
| 其他参数 | TopK k=8, 20 epochs, lr=1e-3 |

### B. Pareto Analysis

**Layer 6** -- MSE 几乎不随 lambda 变化（2.163--2.167），LogitLens Acc 在 lambda<=3e-4 时稳定在 77.57--77.63%，仅 lambda=1e-3 时略降至 77.34%。alignment 单调上升。

| lambda | MSE delta vs baseline | LogitLens delta | Strong% |
|--------|----------------------|-----------------|---------|
| 0 (baseline) | -- | -- | 0.00% |
| 3e-5 | -0.0037 | +0.04pp | 0.00% |
| 1e-4 | -0.0001 | -0.02pp | 21.24% |
| 3e-4 | -0.0011 | -0.02pp | 30.82% |
| 1e-3 | -0.0010 | -0.25pp | 52.51% |

**Layer 11** -- MSE 同样几乎不变（22.25--22.28），LogitLens Acc 在 lambda<=3e-4 时稳定在 87.72--87.77%，lambda=1e-3 时降至 87.30%。alignment 在 lambda=1e-4 时已跳至 87.6% Strong。

## 结论

1. **Anchor loss 几乎是 "free lunch"**: 在 lambda<=3e-4 的范围内，MSE 和 LogitLens Acc 相对 baseline（lambda=0）基本无变化，而 vocab alignment 显著提升。
2. **Layer 11 对 anchor 更敏感**: lambda=1e-4 即可达到 87.6% Strong alignment（接近饱和），而 Layer 6 在 lambda=1e-3 时才达到 52.5% Strong。这与 Layer 11 更接近输出层、其表示本身就更接近 vocab 空间一致。
3. **lambda=1e-3 是拐点**: 两层在此处开始出现可观测的 LogitLens Acc 下降（Layer 6: -0.25pp, Layer 11: -0.46pp），说明 anchor 开始干扰 reconstruction 质量。

### Pareto 最优推荐

- **Layer 6: lambda = 3e-4** -- 以几乎零 MSE/Acc 代价获得 30.8% Strong + 41.9% Medium（共 72.7% 的 feature 有明确 vocab 对应）。若可接受 0.25pp Acc 下降，lambda=1e-3 可进一步提升至 52.5% Strong。
- **Layer 11: lambda = 1e-4** -- 以几乎零 MSE/Acc 代价即可达到 87.6% Strong alignment，已接近饱和。增大 lambda 无显著额外收益，反而在 1e-3 处引入 0.46pp Acc 下降。
